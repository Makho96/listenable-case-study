The content and delivery system to set your podcast apart

“The Bert Show” podcast series owners wanted to see detailed statistics about their show. We needed to create a web application that shows statistics from various social networks like Youtube, Spotify, Megaphone, and Twitter. What they needed to see was how people were reacting to each podcast: how many subscribers they gained(in the case of Youtube), how many subscribers they lost, what was the average listening/view time per episode, which age category was listening to each episode and how are listeners distributed by age category?
Our team began the research and found out that Youtube has an API that gives developers the data we needed for this web application.

Registration and authorization
Authorization and registration are possible with google authorization with “The Bert Show” channel credentials. After the authorization process is finished the user will be redirected to the Dashboard page.
Hover for Details

The demographics widget consists of the USA map and table and shows the user the total views and total listen time of the channel grouped by the states. More views number mean more blue color of the state, fewer views - dark. On each state hover user can see detailed information about views: distributed by age and gender. The table can be sorted with any column user wants: ascending and descending. With the "see all" URL more details can be seen on the Channel stats page.

The general traffic widget shows users channel views daily and monthly. After clicking on "all stats" users are redirected to the Channel stats page for more detailed information.

The popular episodes Widget shows the top 5 most popular episodes and their details like title, views, average listening time, and publish date. The list can be sorted with any available information. After clicking on "see all" users are redirected to the Episodes page for more detailed information.

The popular topics widget shows the most popular 5 tweets in the USA in the last 24 hours. After clicking "see all" users are redirected to the Popular topics page for more detailed information.

Dashboard
The dashboard gives users an overview of the information through the widgets they can take see in The listenable web application. Widgets consist of charts, tables and lists.
Channel Stats
On the channel stats page users can see detailed statistics of “The Bert Show” youtube channel: total listening time based on the USA states, also it is possible to check in which state what age categories were watching/listening the podcasts. Audience statistics divided by gender. The user can see channel traffic data both daily and monthly.
